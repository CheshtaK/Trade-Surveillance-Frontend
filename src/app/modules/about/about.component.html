Project Background Front Running

It is an illegal practice of purchasing a security based on advance non-public information regarding an expected large transaction that will affect the price of a security.

Wash Trade

It is a process whereby a trader buys and sells a security for the purpose of feeding misleading information to the market.


Project Scope Trade surveillance software which performs:

→ Random trade generation

→ Manual trade entry

→ Front running detection

→ Wash Trade detection

→ Graphical representation of market manipulation
